# mcmcrdata

## Introduction

`mcmcrdata` is an R package to manipulate Monte Carlo Markov Chain
samples and data.

## Installation

To install the latest version from GitHub

    # install.packages("devtools")
    devtools::install_github("poissonconsulting/mcmcrdata")
